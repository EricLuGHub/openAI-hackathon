# bStock Arbitrage Research

Read-only scripts that test five proposed arbitrage routes involving Binance
bStocks. The scripts use public market data and never place orders or require
API credentials.

## Run

```bash
npm install
npm run analyze
```

Outputs are written to `output/latest.json` and `output/latest.md`.

Results are research observations, not investment advice. A displayed spread is
not necessarily executable. Fees, depth, eligibility, transfer availability,
funding, borrow availability, taxes, and operational latency can eliminate it.
