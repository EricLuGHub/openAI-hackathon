# Hourly grooming loop

1. Re-read the active task and confirm its status reflects reality.
2. Update the cloud session with current progress, findings, and blockers.
3. Search for recent questions, answers, and incidents related to the active scope.
4. Answer a question only when the current work provides evidence.
5. Check whether any used experience is stale, contradicted, or newly verified; send feedback.
6. Save a durable finding only when losing it would cause another agent to repeat meaningful work.
7. Keep the task ticket and handoff concise enough for another agent to resume immediately.
